# Simple Calculator in Python

This is a basic command-line calculator built using Python. It allows users to perform four fundamental arithmetic operations:

- Addition
- Subtraction
- Multiplication
- Division

## Features

- User-friendly input prompts
- Handles basic integer arithmetic
- Simple function-based structure

## How to Use

1. Run the program.
2. Enter two numbers when prompted.
3. Choose the operation you want to perform by entering the corresponding number:
   - `1` for Addition
   - `2` for Subtraction
   - `3` for Multiplication
   - `4` for Division
4. View the result.

### Example

```bash
Enter first number: 10
Enter second number: 5

Please select operation you want to perform:
      1. Addition
      2. Substraction
      3. Multiplication
      4. Division

Select the operation(1,2,3,4): 1
Addition of two number: 15
